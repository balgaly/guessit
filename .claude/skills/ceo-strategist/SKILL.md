## Feature Prioritization (ICE Framework)
When prioritizing features, score each on:
- **Impact** (1-5): How much does this move the needle for users?
- **Confidence** (1-5): How sure are we this will work?
- **Ease** (1-5): How quickly can we ship this?
- **ICE Score** = Impact × Confidence × Ease

Present results as a ranked table with reasoning.
